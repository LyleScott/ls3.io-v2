Description¶

Partners are created by Gooee and contain their own Users that can create Customers. A Partner is also responsible for a creating many of the Products available to a Customer.

Data Model¶

Key Datatype Notes
id ref read-only
created datetime read-only
modified datetime read-only
name string required, max_length=100, unique
description string  
location location

preferences json  
css url max_length=512
privacy_policy_url url max_length=1000
google_homegraph_api_key string max_length=256, write-only

Complex Constraints¶

Partners with products must have is_manufacturer set to true.

Only Gooee users can change the is_gateway_lite_provider on a Partner.

Partner Devices Total¶

This is the total number of Devices that are associated with a Product of the Partner.

Partner Preferences Model¶

These are portal theme color preferences associated with a partner. They can be included in the request body when creating/updating/patching a partner.

schema

"preferences" : {
    'header-bg-color': '#0E1A35',
    'header-text-color': '#FFFFFF',
    'sidemenu-bg-color': '#15254A',
    'sidemenu-text-color': '#FFFFFF',
    'primary-button-color': '#38AECE',
    'primary-button-text-color': '#FFFFFF',
    'accent-color': '#D7007f',
}

Customer Management¶

A Partner can be granted access to manage a Customer’s resources. For information on how to do this, see Partner Access.

Note that in order to allow Customer to grant access, Partners are visible to all Customers, regardless of scoping.

  • 201 Created –

POST /partners/ HTTP/1.1
Host: api.gooee.io
Content-Type: application/json

{
  "name": "Example Partner",
  "location": {
    "addr1": "7711 East Pleasant Valley Rd.",
    "addr2": "P.O. Box 318010",
    "city": "Cleveland",

    "country": "US"
  },
  "is_manufacturer": true,
  "is_gateway_lite_provider": false,
