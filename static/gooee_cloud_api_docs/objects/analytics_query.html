DescriptionÂ¶

The Analytics Query resource represents a standard analytics history query that is wrapped up into an object that can be executed on demand via the Cloud API or combined with Schedules (coming soon).

Data ModelÂ¶

Key Datatype Notes
id ref read-only
created datetime read-only
modified datetime read-only
created_by ref read-only
name str required, max_length=100
parameters obj JSON Field representing URL parameters to supply to an analytics history query
last_run datetime read-only and updated on every run attempt
last_successful_run datetime read-only and updated on every successful run attempt
tags array(str) max_length=50 per tag
customer_scopes array(ref) see Scope
partner_scopes array(ref) see Scope

Run an Analytics QueryÂ¶

You may supply a parameters object to override parameters on the saved query or to use new parameters.

This is useful for something like a “data-preview” where you want to just get a few results of a saved query. Or, to reduce the friction in experimenting and viewing data in slightly different ways.

Status Codes:
  • 201 Created –
