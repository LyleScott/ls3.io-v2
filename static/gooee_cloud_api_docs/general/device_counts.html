Device Counts¶

Device counts are available on the objects mentioned below. To view the counts, you must add device_counts to the _annotate URL parameter during a GET request. This will add a device_counts field to the response that contains the total and online counts of all child Devices found under that entity. If there are no counts listed for a Device type, this means there are 0 of them.

Objects supporting device counts¶

  • Customer
  • Building
  • Space
  • Device

Counts Annotation API¶

GET /customers?_annotate=device_counts¶

Example response (counts object):

HTTP/1.1 200 OK
Content-Type: application/json

[
  {
    "name": "My Customer",
    "counts": {
      "gateway": {
         "total": 2,
         "online": 1
      },
      "wim": {
         "total": 10,
         "online": 7
      }
    }
  }
]
