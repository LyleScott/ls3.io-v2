API Tokens¶

Requests can be made directly to the Gooee Cloud API through the use of API tokens. Data can be accessed programmatically, rather than relying on visiting the Gooee Cloud Portal.

Note

A default API token is generated automatically for each Customer.

Note

Coming soon: granting each token a specific set of permissions.

Data Model¶

Key Datatype Notes
created datetime read-only
description string required, only visible for Customer API tokens.
api_token string read-only
account_status string read-only
usage json read-only

Permissions and Scoping¶

You must have the edit_customer permission set to manage Customer API Tokens.

You must be scoped to the parent Customer in order to access a Customer API Token instance.

API Endpoints¶

List Customer API Tokens¶

GET /customers/(id)/api_tokens¶
Status Codes:
  • 200 OK –

Example request:

GET /customers/93378718-b2ad-4b6c-a1bb-39c5436bf3e6/api_tokens HTTP/1.1
Host: api.gooee.io
Content-Type: application/json

Example response:

HTTP/1.1 200 OK
Content-Type: application/json

    "created": "2016-11-09T04:04:07.007725Z",

Create a Customer API Token¶

POST /customers/(id)/api_tokens¶
Status Codes:
  • 201 Created –

Example request:

POST /customers/93378718-b2ad-4b6c-a1bb-39c5436bf3e6/api_tokens HTTP/1.1
Host: api.gooee.io
Content-Type: application/json

{
  "description": "testing api token"
}

Example response:

HTTP/1.1 201 OK
Content-Type: application/json


{
  "created": "2020-03-13T20:19:11.869809Z",
  "description": "testing api token",
  "api_token": "24c355c8ed563c0075371059d2a1057fe38b3919fff13bdebfd3007317fe8408b40ed340",
  "account_status": "active",
  "usage": "This feature is unavailable at this time."
}

Activate a Customer API Token¶

POST /customers/(id)/api_tokens/{api_token}/activate¶
Status Codes:
  • 204 No Content –

Deactivate a Customer API Token¶

POST /customers/(id)/api_tokens/{api_token}/deactivate¶
Status Codes:
  • 204 No Content –

Delete a Customer API Token¶

DELETE /customers/(id)/api_tokens/{api_token}¶
Status Codes:
  • 201 Created –

View User API Token¶

GET /users/(id)/api_token¶
Status Codes:
  • 200 OK –
