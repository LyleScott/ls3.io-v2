Search¶

The Search API provides site-wide, case-insensitive, locale-insensitive fulltext searching capabilities. See Parameters for how to query.

Data Model¶

Key Data Type Notes
model string Model representing the record.
id ref UUID of the record.
relevance number Relevance of the search result based on your query parameter.
fields object Fields of the result. See Supported Model Fields.

Supported Model Fields¶

Depending on the model of the search result the fields provided will be different. When conducting a search these are the fields for each model that are searched and returned in the result:

  • Area Fields
  • Building Fields
  • Connected Product Fields
  • Customer Fields
  • Device Fields
  • Event Rule Fields
  • Partner Fields
  • Product Fields
  • Property Rule Fields
  • Scene Fields
  • Schedule Fields
  • Space Fields
  • User Fields
  • User Group Fields

Parameters¶

The following table lists all available search parameters, which are supplied in the URL query string.

Note

Fuzzy searching is not yet supported. This will be implemented in the future.

Filter Description Example
q Some text to search for. Minimum of 3 characters. /search?q=foobar
models A list of models to search. If not given, all models are searched /search?q=quux&models=customer,building
