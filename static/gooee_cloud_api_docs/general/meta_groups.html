Description¶

Devices, Products and Connected Products support the ability of grouping their meta values with pre-configured groups that can be used later for analytics and permissions purposes. By default there are five grouping values defined: ~product_performance, ~motion, ~energy, ~admin, ~protected.

Note

Groups starting with the ~ character are system groups and can only be assigned to Meta values by Gooee users. Meta with these Groups may only be deleted by Gooee Users.

Note

Meta in the ~protected group is governed by Gooee and has pre-defined JSON schema that it’s value must validate against.

Data Model¶

This is the data model used to configure the Meta Groups:

Key Datatype Notes
id ref read-only
created datetime read-only
modified datetime read-only
name string required, max_length=255 Only lowercase alphanumeric characters, underscores and ~ are allowed.
display_name datetime read-only, auto generated from name.
is_system_group boolean read_only
customer_scopes array(ref) see Scope
partner_scopes array(ref) see Scope
